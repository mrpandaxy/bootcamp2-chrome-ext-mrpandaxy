Motivador Diário

Extensão para Google Chrome (Manifest V3) — projeto do Bootcamp II.

O que faz?

Mostra uma mensagem motivacional diferente toda vez que você clica no botão do popup. Simples, leve e inspirador!

Como instalar

  1. Baixe o ZIP da extensão ou clone o repositório.
  2. Acesse chrome://extensions no Chrome.
  3. Ative o modo desenvolvedor.
  4. Clique em "Carregar sem compactação" e selecione a pasta do projeto.
  5. Clique no ícone da extensão para usar!
